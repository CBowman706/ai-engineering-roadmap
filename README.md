# 🤖 AI Engineering Roadmap Tracker

A personal progress tracker built to document my 6-month journey toward becoming an AI/ML Engineer. Tracks weekly learning progress, projects, and milestones — all in a single deployable HTML file.

🔗 **[Live Demo](https://ai-security-tracker-clt.netlify.app/)**

---

## 🎯 What It Does

- Tracks **weekly learning progress** across a 6-month AI/ML Engineering roadmap
- Logs **projects and milestones** completed along the way
- Provides a visual overview of progress at a glance
- Single file — no dependencies, no build step, runs anywhere

---

## 🛠️ Built With

- **HTML / CSS / JavaScript**
- Zero dependencies — runs in any browser
- Deployed via Netlify

---

## 🚀 Getting Started

### Run Locally
```bash
git clone https://github.com/CBowman706/ai-engineering-roadmap.git
cd ai-engineering-roadmap
open index.html
```

Or just open `index.html` directly in your browser.

### Deploy Your Own
1. Fork this repo
2. Drag the `index.html` file into [Netlify Drop](https://app.netlify.com/drop)
3. Your tracker is live instantly
