✓
Google "sample Apache access log" — study what a real security log looks like
Look for: IP addresses, timestamps, HTTP methods (GET/POST), status codes (401=unauthorized, 404=not found, 500=server error). These are what analysts scan for threats.
✓
Create a sample log file on your Pi to practice with
cd ~/ai-security cat > sample.log << 'EOF' 192.168.1.100 - - [22/Mar/2026:10:00:01] "GET /login HTTP/1.1" 200 10.0.0.5 - - [22/Mar/2026:10:00:45] "POST /admin HTTP/1.1" 401 10.0.0.5 - - [22/Mar/2026:10:00:46] "POST /admin HTTP/1.1" 401 10.0.0.5 - - [22/Mar/2026:10:00:47] "POST /admin HTTP/1.1" 401 192.168.1.200 - - [22/Mar/2026:10:01:00] "GET /index.html HTTP/1.1" 200 EOF
This creates a fake log with a suspicious brute-force pattern (same IP, 3 failed logins)

Build the AI Log Analyzer

✓
Type in the full script
import ollama from datetime import datetime # Read the log file with open('sample.log', 'r') as f: log_data = f.read()[:3000] # first 3000 chars print("🔍 Analyzing log file with AI...") print("-" * 40) # Ask AI to analyze it as a security analyst would response = ollama.chat(model='phi3', messages=[{ 'role': 'user', 'content': f"""You are a cybersecurity analyst. Analyze this security log and identify: 1. Suspicious IP addresses 2. Failed login attempts (401/403 errors) 3. Unusual patterns or potential attacks 4. Risk assessment: Low / Medium / High LOG DATA: {log_data}""" }]) timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S") print(f"📋 AI Security Analysis [{timestamp}]:") print(response['message']['content'])

›

✓
Build the full menu-driven version
import ollama from datetime import datetime def analyze_log(): f = input("Log file (Enter=sample.log): ") or "sample.log" try: with open(f) as file: data = file.read()[:3000] r = ollama.chat(model='phi3', messages=[{ 'role':'user', 'content':f'Analyze this security log for threats:\n{data}' }]) print(f"\n📋 [{datetime.now().strftime('%H:%M')}]") print(r['message']['content']) except FileNotFoundError: print(f"❌ File not found: {f}") def ask_question(): q = input("Security question: ") r = ollama.chat(model='phi3',messages=[{'role':'user','content':q}]) print("\n🤖", r['message']['content']) while True: print("\n🛡️ AI Security Assistant") print("1. Analyze a log file") print("2. Ask a security question") print("3. Exit") c = input("Choose (1-3): ").strip() if c=='1': analyze_log() elif c=='2': ask_question() elif c=='3': print("Stay secure. 👋"); break else: print("Invalid — try again")
